# Client - simple clients app

* mongodb + nodejs 9+

## Install
* create config file server/app/config.js from server/app/config.sample.js
* write you db setting
* cd server
* npm run install_app
* npm run start

## Run
* go to http://localhost:3005 for app
* go to http://localhost:3005/api-docs/ for api docs
* DONE
